# راهنمای تنظیم ایمیل تأیید هویت

## حالت فعلی (بدون تنظیم)

اپلیکیشن **همین الان** کار می‌کند. نیازی به تنظیم ایمیل نیست.

وقتی کاربر وارد می‌شود، کد تأیید:
1. ✅ **در مرورگر نشان داده می‌شود** (جعبه سبز رنگ در صفحه تأیید)
2. ✅ **در terminal نشان داده می‌شود** (در کنسول `npm run dev`)

---

## فعال‌سازی ایمیل واقعی (Gmail)

### مرحله ۱ — ورود به اکانت گوگل
به آدرس زیر بروید:
```
https://myaccount.google.com/security
```
مطمئن شوید **تأیید هویت ۲ مرحله‌ای** فعال است.

### مرحله ۲ — ساخت App Password
به آدرس زیر بروید:
```
https://myaccount.google.com/apppasswords
```
- نام: `CodeAssist`
- یک رمز ۱۶ کاراکتری دریافت می‌کنید مثل: `abcd efgh ijkl mnop`

### مرحله ۳ — تنظیم .env.local
فایل `.env.local` را باز کنید و این دو خط را uncomment کنید:

```env
SMTP_USER=your_gmail@gmail.com
SMTP_PASS=abcd efgh ijkl mnop
```

### مرحله ۴ — ری‌استارت
```bash
npm run dev
```

## جریان احراز هویت

```
/register   →  ثبت‌نام
    ↓
/login      →  ایمیل + رمز عبور
    ↓
            →  کد ۶ رقمی (ایمیل یا terminal یا مرورگر)
    ↓
/chat       →  داشبورد چت‌بات
```

## اطلاعات فنی

- **انقضای کد:** ۱۰ دقیقه
- **محدودیت ارسال مجدد:** ۱ دقیقه
- **ذخیره‌سازی:** رمزنگاری شده در MongoDB (`verifyCode`, `verifyExpiry`)
- **وابستگی خارجی:** صفر (از Node.js داخلی استفاده می‌شود)
